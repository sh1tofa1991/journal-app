# Journal App

Консольное приложение для учета успеваемости студентов группы.

## Функционал
- Добавление студентов с оценками
- Просмотр списка всех студентов
- Фильтрация по среднему баллу
- Изменение оценок
- Статистика группы

## Требования
- Go 1.21 или выше

## Запуск
```bash
go mod init journal_app
go run .
